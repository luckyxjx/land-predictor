#  Land Price Predictor - PySpark Edition

Big Data land price prediction using Apache Spark distributed computing.

## Features
-  **PySpark ML Pipeline** - Distributed machine learning
-  **Big Data Processing** - Handle millions of records
-  **Real-time Predictions** - In-memory computing
-  **Scalable Architecture** - Grows with data size

## Quick Start
```bash
pip install -r requirements.txt
streamlit run land_predictor_spark.py
```

## Spark Advantages
- 100x faster than traditional systems
- Distributed processing across clusters
- In-memory analytics
- Real-time big data ML
