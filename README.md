#  Land Price Consultant

land price prediction for all major Indian cities.

## Quick Start
```bash
pip install -r requirements.txt
streamlit run land_dealer.py
```

## Features
-  All India coverage (8 states, 20+ cities)
-  Instant price predictions
-  Future value forecasts
-  Google Maps-style location selection

## Usage
1. Select State → City → Area
2. Enter plot details
3. Get instant price estimate
4. View future projections
